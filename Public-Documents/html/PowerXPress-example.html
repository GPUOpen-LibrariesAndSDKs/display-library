ADL (AMD Display Library) Public APIs  Version 17.0
PowerXPress

Description: This sample demonstrates the usage of PowerXpress APIs. Usage model includes listing application which are classified as Most Recent Used, Discrete, Integrated, profiled and total Applications.

Supported OS: Windows

Init and Destroy: Initialize the library before using it and destroy it after the usage. Refer to InitADL() for initialization and clean-up is done inside main function only.

Functionality:
  • Check if the platform is PowerXpress/Hybrid Graphics supported.
  • Read the applications via “ADL2_ApplicationProfiles_HitListsX3_Get” with different list types as input parameter.

Sample Path: Sample\PowerXpress
