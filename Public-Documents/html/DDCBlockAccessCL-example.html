ADL (AMD Display Library) Public APIs  Version 17.1
DDCBlockAccessCL

Description: This command line sample demonstrates accessing DDC (Display Data Channel) using I2C communication. Provides a user interface to allow read and write Luminance, Contrast and Color Temperature features.

Supported OS: Windows

Init and Destroy: Initialize the library before using it and destroy it after the usage. Refer to InitADL() for initialization and clean-up is done inside main function only.

Functionality:
  • User interface lists the adapter and display names along with indexes.
  • Access to DDC block via “ADL_Display_DDCBlockAccess_Get”.
  • Allows the user to select different features such as Luminance, Contrast and Color Temperature for read and write to DDC.

Sample Path: Sample\DDCBlockAccessCL
