ADL (AMD Display Library) Public APIs  Version 17.1
ColorCaps

Description: This sample provides the usage model of Color APIs. Usage model includes reading Display Color capability. If capable increases, decreases and restores the brightness of each display.

Supported OS: Windows

Init and Destroy: Initialize the library before using it and destroy it after the usage. Refer to InitADL() for initialization and clean-up is done inside main function only.

Functionality:
  • Identify a connected display on each adapter via “ADL_Display_DisplayInfo_Get” and read caps using “ADL_Display_ColorCaps_Get”
  • Identify caps support for “ADL_DISPLAY_COLOR_BRIGHTNESS” and if yes, read the range of the feature with “ADL_Display_Color_Get”. We can also use this for “ADL_DISPLAY_COLOR_CONTRAST”, “ADL_DISPLAY_COLOR_SATURATION” & “ADL_DISPLAY_COLOR_HUE” features
  • Increase, decrease and restore the brightness using “ADL_Display_Color_Set”.

Sample Path: Sample\ColorCaps
