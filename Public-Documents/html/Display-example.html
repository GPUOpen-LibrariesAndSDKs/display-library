ADL (AMD Display Library) Public APIs  Version 17.0
Display

Description: This sample provides the usage model of Display APIs. Usage model includes read & write Virtual Super Resolution and write Integer Scaling.

Supported OS: Windows

Init and Destroy: Initialize the library before using it and destroy it after the usage. Refer to InitADL() for initialization and clean-up is done inside main function only.

Functionality:
  • Identifies connected displays on a given system.
  • Argument “g” list the displays with Virtual Super Resolution state (On/Off).
  • Argument “s” applies Virtual Super Resolution for all the displays. On/Off state is read from second argument.
  • Argument “si” applies Integer Scaling for all the displays. On/Off state is read from second argument.
  • Argument “l” applies get display connectivty experience settings for all the displays.

Sample Path: Sample\Display
