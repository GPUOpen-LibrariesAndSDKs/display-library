ADL (AMD Display Library) Public APIs  Version 17.0
SmartDC

Description: This sample demonstrates how to configure SmartDC.

Supported OS: Windows

Init and Destroy: Initialize the library before using it and destroy it after the usage. Refer to InitADL() for initialization and clean-up is done inside main function only.

Functionality:
  • Allow to get the capability of SmartDC on current system using "ADL2_SmartDC_Caps".
  • Allow to get current status of SmartDC using "ADL2_SmartDC_Status_Get".
  • Allow to enable/disable SmartDC using "ADL2_SmartDC_Status_Set".

Sample Path: Sample\SmartDC
